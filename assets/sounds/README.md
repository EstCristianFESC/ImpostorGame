# Sound Files Setup

Para los efectos de sonido, necesitas agregar estos archivos en la carpeta `assets/sounds/`:

## Archivos Requeridos:

1. **reveal.mp3** - Sonido al revelar palabra

   - Sugerencia: Un "whoosh" o "reveal" corto (1-2 segundos)
   - Fuente gratuita: https://mixkit.co/free-sound-effects/whoosh/

2. **suspense.mp3** - Sonido de suspenso durante countdown
   - Sugerencia: Ambiente tenso, loop corto (3-5 segundos)
   - Fuente gratuita: https://mixkit.co/free-sound-effects/suspense/

## Alternativa Temporal:

He configurado el código para usar URLs de Mixkit como fallback. Si no agregas los archivos locales, el juego usará sonidos desde internet (requiere conexión).

## Cómo Agregar Tus Propios Sonidos:

1. Descarga los archivos MP3
2. Colócalos en `assets/sounds/`
3. Asegúrate de que se llamen exactamente:
   - `reveal.mp3`
   - `suspense.mp3`

¡Listo! El código ya está preparado para usarlos.
